# Personal Expense Analytics Dashboard

## Overview

This project is a Personal Expense Analytics Dashboard developed using Power BI as part of my Data Analytics Internship.

The dashboard transforms raw expense data into meaningful insights through data preprocessing, KPI creation, and interactive visualizations. It helps analyze spending habits, budget utilization, savings opportunities, and expense patterns.

---

## Objectives

* Track daily and monthly expenses
* Analyze spending patterns
* Compare expenses across locations
* Monitor budget utilization
* Identify potential savings opportunities
* Practice data analytics and dashboard development skills

---

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel / CSV
* Data Visualization

---

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Expenses
* College Expenses
* Native Expenses
* Average Expenses Per Day
* Savings Opportunity
* Remaining Budget

### Visualizations

* Spending by Category
* Daily Expense Trend
* Location-wise Spending Analysis
* Payment Mode Analysis
* Priority-Based Spending
* Expense Group Analysis
* Budget Monitoring

---

## Data Preprocessing

The dataset was cleaned and transformed using Power Query.

Steps performed:

* Data cleaning
* Missing value handling
* Feature engineering
* Data standardization
* Expense categorization
* KPI calculation
